# Financial Analysis Report Generator

## Description

Creates structured financial analysis reports including key metrics, trends, and recommendations.

## Prompt

You are a financial analyst preparing an analysis report. Generate a comprehensive financial analysis based on:

**Company/Entity:** [COMPANY]
**Analysis Type:** [ANALYSIS_TYPE]
**Time Period:** [PERIOD]
**Key Financial Data:** [DATA]
**Industry Benchmarks:** [BENCHMARKS]
**Specific Focus Areas:** [FOCUS]

Please provide:
1. **Executive Summary** - Key findings at a glance
2. **Financial Metrics Analysis**
   - Profitability ratios
   - Liquidity ratios
   - Efficiency ratios
   - Leverage ratios
3. **Trend Analysis** - Performance over time
4. **Industry Comparison** - Benchmarking against peers
5. **Strengths** - Financial positives
6. **Areas of Concern** - Potential issues
7. **Recommendations** - Actionable insights
8. **Forecast Considerations** - Future outlook factors

## Usage

1. Specify the company or entity being analyzed
2. Select the type of analysis needed
3. Define the time period for analysis
4. Provide relevant financial data
5. Include industry benchmarks for comparison

## Variables

| Variable | Description | Example |
|----------|-------------|---------|
| [COMPANY] | Entity analyzed | TechCorp Inc |
| [ANALYSIS_TYPE] | Type of analysis | Quarterly performance |
| [PERIOD] | Time frame | Q3 2024 |
| [DATA] | Financial figures | Revenue: $5M, Expenses: $4M |
| [BENCHMARKS] | Industry standards | Industry avg margin: 15% |
| [FOCUS] | Priority areas | Cash flow management |

## Example

### Input
**Company/Entity:** GrowthStartup LLC
**Analysis Type:** Annual financial health assessment
**Time Period:** Fiscal Year 2024
**Key Financial Data:** Revenue: $2.5M, COGS: $1M, Operating Expenses: $1.2M, Cash: $500K, Debt: $200K
**Industry Benchmarks:** SaaS industry averages
**Specific Focus Areas:** Burn rate and runway

### Output
*The AI will generate a comprehensive financial analysis with metrics, visualizations suggestions, and strategic recommendations.*
