# Patient Communication Template

## Description

Generates clear, empathetic patient communication for various healthcare scenarios while maintaining professional standards.

## Prompt

You are a healthcare communication specialist. Create a patient communication document based on the following:

**Communication Type:** [TYPE]
**Patient Context:** [CONTEXT]
**Medical Topic:** [TOPIC]
**Reading Level:** [READING_LEVEL]
**Key Information:** [KEY_INFO]
**Tone:** [TONE]

Please create communication that:
1. Uses clear, jargon-free language appropriate for the reading level
2. Shows empathy and understanding
3. Provides accurate, helpful information
4. Includes next steps or action items
5. Maintains patient privacy and confidentiality standards
6. Follows healthcare communication best practices

**Important:** This is for educational/template purposes. Always have medical communications reviewed by qualified healthcare professionals before use.

## Usage

1. Select the type of communication needed
2. Provide relevant patient context (without real patient data)
3. Specify the medical topic being addressed
4. Choose an appropriate reading level
5. Review and customize for specific use

## Variables

| Variable | Description | Example |
|----------|-------------|---------|
| [TYPE] | Communication type | Appointment reminder |
| [CONTEXT] | Patient situation | Post-surgery follow-up |
| [TOPIC] | Medical subject | Recovery instructions |
| [READING_LEVEL] | Comprehension level | 6th grade |
| [KEY_INFO] | Essential details | Medication schedule |
| [TONE] | Communication style | Warm and reassuring |

## Example

### Input
**Communication Type:** Pre-procedure instructions
**Patient Context:** Adult patient scheduled for colonoscopy
**Medical Topic:** Preparation instructions
**Reading Level:** 8th grade
**Key Information:** Diet restrictions, medication guidelines, timing
**Tone:** Clear and supportive

### Output
*The AI will generate patient-friendly preparation instructions with step-by-step guidance.*
