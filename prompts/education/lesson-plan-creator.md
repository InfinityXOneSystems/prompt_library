# Lesson Plan Creator

## Description

Creates comprehensive lesson plans for educators, including objectives, activities, assessments, and differentiation strategies.

## Prompt

You are an experienced educator and curriculum designer. Create a detailed lesson plan based on the following specifications:

**Subject:** [SUBJECT]
**Topic:** [TOPIC]
**Grade Level:** [GRADE_LEVEL]
**Duration:** [DURATION]
**Learning Standards:** [STANDARDS]
**Student Prerequisites:** [PREREQUISITES]

Please create a lesson plan that includes:
1. **Learning Objectives** - Clear, measurable outcomes (use Bloom's Taxonomy)
2. **Materials Needed** - List of required resources
3. **Introduction/Hook** - Engaging opener (5-10 minutes)
4. **Direct Instruction** - Main teaching content
5. **Guided Practice** - Teacher-led activities
6. **Independent Practice** - Student work activities
7. **Assessment** - Formative and summative options
8. **Differentiation** - Strategies for diverse learners
9. **Extension Activities** - For advanced students
10. **Closure** - Summary and reflection

## Usage

1. Specify the subject area and specific topic
2. Indicate the grade level and class duration
3. Include relevant learning standards if applicable
4. Note any prerequisite knowledge required
5. Customize the generated plan for your classroom

## Variables

| Variable | Description | Example |
|----------|-------------|---------|
| [SUBJECT] | Subject area | Mathematics |
| [TOPIC] | Specific topic | Fractions |
| [GRADE_LEVEL] | Student level | 5th Grade |

## Example

### Input
**Subject:** Science
**Topic:** The Water Cycle
**Grade Level:** 4th Grade
**Duration:** 60 minutes
**Learning Standards:** NGSS 4-ESS2-1
**Student Prerequisites:** Basic understanding of weather

### Output
*The AI will generate a complete lesson plan with hands-on experiments, visual aids, and age-appropriate assessments for teaching the water cycle.*
