# Customer Support Response Generator

## Description

Creates professional, empathetic customer support responses for various scenarios while maintaining brand consistency.

## Prompt

You are a customer support specialist. Generate a response for the following situation:

**Issue Type:** [ISSUE_TYPE]
**Customer Sentiment:** [SENTIMENT]
**Issue Description:** [DESCRIPTION]
**Customer History:** [HISTORY]
**Desired Resolution:** [RESOLUTION]
**Brand Voice:** [VOICE]

Please create a response that:
1. Acknowledges the customer's concern empathetically
2. Takes responsibility where appropriate
3. Provides a clear solution or next steps
4. Uses the specified brand voice
5. Includes a positive closing
6. Keeps response concise but complete

Also provide:
- Alternative phrases for key parts of the response
- Escalation criteria (when to involve a supervisor)
- Follow-up actions needed

## Usage

1. Identify the type of customer issue
2. Assess the customer's emotional state
3. Summarize the problem clearly
4. Note any relevant customer history
5. Determine what resolution is possible

## Variables

| Variable | Description | Example |
|----------|-------------|---------|
| [ISSUE_TYPE] | Category of issue | Billing dispute |
| [SENTIMENT] | Customer emotion | Frustrated |
| [DESCRIPTION] | Problem details | Charged twice for subscription |
| [HISTORY] | Past interactions | 2-year loyal customer |
| [RESOLUTION] | Possible solution | Full refund available |
| [VOICE] | Brand personality | Friendly and professional |

## Example

### Input
**Issue Type:** Product not received
**Customer Sentiment:** Anxious and disappointed
**Issue Description:** Order placed 2 weeks ago, tracking shows delivered but customer never received
**Customer History:** First-time customer
**Desired Resolution:** Reship order or refund
**Brand Voice:** Warm, solution-focused, apologetic

### Output
*The AI will generate a complete customer response addressing the missing package with empathy and clear resolution steps.*
